Login to the MariaDB database console using the commands below:

sudo mysql -u root -p

list of databases:

show databases;

Then create a database called magentodb:
Replace magentodb to your datase name

CREATE DATABASE magentodb;

Next, create a database user called magentodbuser and set a password

CREATE USER 'magentodbuser'@'localhost' IDENTIFIED BY 'new_password_here';

Then grant the user full access to the database

GRANT ALL ON magentodb.* TO 'magentodbuser'@'localhost' WITH GRANT OPTION;

Finally, save your changes and exit.

FLUSH PRIVILEGES;
EXIT;

Create Folder:

mkdir coeandco

At this point, Magento is ready to be downloaded and installed. Use the commands below to download the latest version of Magento. To get Magento’s latest release you may want to use the GitHub repository… Install Composer, Curl and other dependencies to get started…

sudo apt install curl git
curl -sS https://getcomposer.org/installer | sudo php -- --install-dir=/usr/local/bin --filename=composer
